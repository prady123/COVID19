# INTRODUCTION
COVID-19 Classification using Xray images

Posterioanterior x-ray images are used to diagnose whether the person has covid19 or not. VGG16 pre trained model i sused for transfer learning.

# Dataset
The images for covid19 positive are taken from the following repo
https://github.com/ieee8023/covid-chestxray-dataset

and the normal images are sampled from a kaggle dataset for pneumonia
https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia
